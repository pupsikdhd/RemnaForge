# RemnaForge 🛠️

**RemnaForge** (vless-proxy) — это современная панель управления и веб-сервис для администрирования прокси-нод, построенный на базе Next.js 16 и Turbopack. Проект ориентирован на высокую производительность, безопасность и цифровую минимизацию.

## 🚀 Стек технологий

* **Фреймворк:** [Next.js 16](https://nextjs.org/) (App Router, Turbopack, Standalone mode)
* **База данных:** [Prisma ORM](https://www.prisma.io/) + SQLite (локальное, легковесное хранилище)
* **Аутентификация:** [Lucia Auth](https://lucia-auth.com/) (v3) + [WebAuthn / Passkeys](https://simplewebauthn.dev/) (аппаратные ключи YubiKey, Биометрия)
* **Стилизация:** Tailwind CSS v4
* **Управление процессами:** PM2
* **Веб-сервер / Reverse Proxy:** Caddy (автоматический HTTPS/SSL)
